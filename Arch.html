1.2 更新系统时间

timedatectl set-ntp true
timedatectl status

1.3 硬盘分区

1.3.1 查询

# 查询硬盘空间
fdisk -l

# 查询是否分区成功
lsblk

1.3.2 分配磁盘空间

#EFI分区1G, swap分区分配8G, 剩余分配给Linux
#图形化分配磁盘
cfdisk /dev/sda

1.3.3 格式化分区

# 以下 sdax 分别代表分区, 例如 sda1, sda2

# EFI 分区格式化
mkfs.fat /dev/sdax

# linux分区格式化
mkfs.ext4 /dev/sdax

# 创建swap分区
mkswap /dev/sdax
# 刷新swap分区
swapon /dev/sdax

1.3.4 挂载分区

# 挂载Linux分区
mount /dev/sdax /mnt

# 挂载 EFI 分区
mkdir /mnt/efi              
mount /dev/sdax /mnt/efi    #挂载EFI分区

1.4 安装最小系统

1.4.1 换源

# 自动换源: 手动取消清华源注释
sudo wget -O /etc/pacman.d/mirrorlist https://www.archlinux.org/mirrorlist/all/

# 手动换源: 编辑 /etc/pacman.d/mirrorlist 文件, 添加下面三行
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinux/$repo/os/$arch
Server = http://mirrors.aliyun.com/archlinux/$repo/os/$arch

1.4.2 安装基本套件

pacstrap /mnt base linux linux-firmware

1.5 生成 fstab 文件

# 生成 fstab 文件
genfstab -U /mnt >> /mnt/etc/fstab

#检查是否生成 fstab 文件
cat /mnt/etc/fstab

1.6 进入安装的系统

# 进入安装的系统, 如果出现差错, 应重新进行硬盘分区
arch-chroot /mnt

# 设置上海时区, 创建软链接
ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

# 更新硬件时间   
hwclock --systohc

2.4 DWM

  1. 在所有的 suckless 的软件中, 都需要修改 config.mk 文件中的 X11INC 和 X11LIB 为 

    X11INC = /usr/include/X11
X11LIB = /usr/lib/X11
  2. 在 make 了一次 dwm 之后, 把 config.deb.h 删除, 要不然会和自己的配置冲突
  3. 打补丁: patch < 补丁.diff
  4. 补丁推荐:
    1. autostart
      1. 自启动脚本文件: ~/.dwm/autostart.sh, 并且要为它添加可执行权限: chmod + x autostart.sh, 才能自动执行
  5. 修改 dmenu 为 rofi: {"rofi", "-show", "run", NULL }; 注意 NULL后面有空格
  6. 修改 termcmd 中的 st 为 alacritty
  7. 修改字体为 Source Code Pro
  8. 把字符 []= 等修改为: 平铺, 浮动, 单页
  9. 快捷键参考链接
  10. 由于 dwm 和 emacs 的键位冲突, 把 dwm 的主键改为 win 键
  11. 截图快捷键: static const char* flameshot[] = { "flameshot", "gui", NULL };
  12. 在 dwm.c 中注释掉 [EnterNotify] = enternotify, 使得不要让鼠标获取焦点
  13. 修改状态栏配色
    1. 按照文章的配色修改: 配色修改
    2. 修改 config.h 中的 borderpx 为 0

2.10 ranger

  1. 复制配置文件到 home.config 目录: ranger --copy-config=all
  2. 查看隐藏文件: 修改 rc.conf 的 show_hidden 为 true
  3. 及时预览图片: 修改 rc.conf 的 preview_images 为 true, 修改 preview_images_method 为 ueberzug
  4. 修改默认编辑器为 nvim: 修改 rifle.conf 的两处的 ${VISUAL:-$EDITOR} 为 nvim
  5. 按照 wiki 中的提示, 修改默认的文件打开方式:
    1. word, excel, ppt
  6. 在 rc.conf 中修改 dT 命令: map dT shell trash-put %s.
  7. 在 rc.conf 中修改 vcsaware, 从而可以看到 git 状态

  8. 添加图标 

    git clone https://github.com/alexanderjeurissen/ranger_devicons ~/.config/ranger/plugins/ranger_devicons
echo "default_linemode devicons" >> $HOME/.config/ranger/rc.conf
  9. 预览
    1. 视频: 在 scope.sh 文件下, 取消注释 video 的批注, 并且要下载安装 ffmpegthumbnailer
    2. pdf: 在 scope.sh 文件下, 取消注释 pdf 的批注
  10. 行号
    1. 在 rc.conf 下, 修改 line-number relative, 显示动态的相对行号
    2. 在 rc.conf 下, 修改 oneinidexed 为 true, 行号从 1 开始

2.11 picom

  1. 复制配置文件: cp /etc/xdg/picom.conf ~/.config/picom.conf
  2. 添加自启动: picom -b 添加到 ~/.xinitrc 的 exec dwm 之前
  3. 注释掉 picom.conf 中的 Fading

  4. 屏幕会有撕裂现象, 对于 AMD 显卡, 在 /etc/X11/xorg.conf.g 目录下, 新建一个 20-amdgpu.conf 的, 写入下面配置: 

    Section "Device"
Identifier "AMD"
Driver "amdgpu"
Option "TearFree" "true"
EndSection

    重启, 即可解决屏幕撕裂问题

  5. xprop 可以显示软件的名称

2.12 proxychains

  1. 在配置文件中写下: 

    export http_proxy="socks5://127.0.0.1:1089"
export https_proxy="socks5://127.0.0.1:1089"

  2. 它无法让 yay 走代理, 如果要 yay 走代理, 要使用下面的命令 

    export http_proxy=http://127.0.0.1:8889
# 这里用的是 http, 也可以用 socks

  3. 也可以用 alias 来快速运行上面命令 

    alias setproxy="export http_proxy=http://127.0.0.1:8889"
alias unsetproxy="unset http_proxy"

2.13 设置中文

  1. 如果不设置中文, 会出现以下问题:
    1. dwm 状态栏会出现 broken
    2. st 会打不开
    3. fcitx5 会无法在 emacs 中打开
    4. google 浏览器等应用默认不是中文
  2. 设置中文: 在 ~/.xinitrc 中加入 export LANG=zh-CN.UTF-8 和 export LANGUAGE=zh_CN:en_US
